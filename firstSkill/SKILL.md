# 日报 Skill

## 用途

帮助用户快速撰写每日工作日报

## 调用方式

`/日报` 或 `/daily-report`

## 执行步骤

1. **问候用户**

   - 友好地询问今日工作内容

2. **收集信息**

   - 询问今日完成了哪些工作
   - 询问明日计划
   - 询问是否有问题或感想（可选）

3. **生成日报**

   - 使用以下格式输出：

   ```markdown
   ## 📅 日期：YYYY-MM-DD
   
   ### 今日完成
   - ...
   
   ### 明日计划
   - ...
   
   ### 备注/问题
   - ...
   ```

## 输出要求

- 使用中文
- 简洁明了
- bullet points 清晰
- 最后输出 "task completed"
